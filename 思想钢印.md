### 9️⃣ 动态 AI 台词层添加
**时间**: 2026年3月22日 晚间  
**文件**: `src/app/pages/HardwareControl.tsx`  
**修改内容**:
- ✅ **创建台词库配置** `phaseVoiceLines` 对象
  - `preheat` (预热阶段): '现在只是开始……', '再靠近一点', '慢慢来，我在感受你'
  - `ongoing` (进行中): '别停……', '再慢一点', '继续，我喜欢这样'
  - `preclimax` (高潮前奏): '就是这样……', '不要停下来', '更深一点，我还想要'
  - `climax` (巅峰时刻): '就是这样……', '不要停下来', '更深一点，我还想要'
  - `afterglow` (余韵回荡): '还没结束……', '再陪我一会儿', '记住刚才的感觉'
- ✅ **添加状态管理**:
  - `currentVoiceLineIndex` 跟踪当前台词索引
  - 阶段改变时自动随机切换台词
- ✅ **UI 设计**:
  - 位置：语音波形上方，产品设备下方
  - 样式：深色半透明圆角卡片（`from-purple-900/40 to-pink-900/30`）
  - 边框：紫色发光（`border-purple-500/30`，hover 时加强）
  - 文本：紫→粉渐变色，居中显示
  - 发光效果：`shadow-lg shadow-purple-500/10`
- ✅ **交互逻辑**:
  - 播放时：`opacity-100`（活跃态）
  - 暂停时：`opacity-40`（弱化态）
  - 过渡时间：500ms 平滑切换
- 🎯 **目标**: 升级用户体验，从纯"硬件控制页"改造为"有角色情绪驱动的硬件体验页"

### 🔟 音频系统集成（Web Audio API + Web Speech API）
**时间**: 2026年3月22日 深夜  
**文件**: `src/app/utils/audioManager.ts`（新建）、`src/app/pages/HardwareControl.tsx`  
**修改内容**:
- ✅ **创建 AudioManager 类**
  - 管理 Web Audio API 音效生成
  - 自动初始化 AudioContext（响应用户交互）
  - 支持 context suspended 状态恢复
- ✅ **实现三种音效**:
  - `playPlaySound()` - 播放时的上升音（400Hz→800Hz）
  - `playPauseSound()` - 暂停时的下降音（600Hz→300Hz）
  - `playPhaseChangeSound()` - 阶段切换时的清晰音（800Hz→1200Hz）
- ✅ **AI 语音朗读**:
  - `speakVoiceLine(text)` - 使用 Web Speech API
  - 中文语音合成，支持 zh-CN
  - 自动调节语速、音调、音量
  - 支持 onstart、onerror 回调监听
- ✅ **错误处理**:
  - 所有音效方法都包装在 try-catch 中
  - 捕获 AudioContext 初始化失败
  - 浏览器不支持时优雅降级（打印警告）
- ✅ **集成到控制页**:
  - 播放按钮点击时播放"开始"音效 + 朗读台词
  - 暂停按钮点击时播放"暂停"音效 + 停止语音
  - 阶段切换时播放"阶段"音效 + 朗读新台词
- 🎯 **目标**: 让应用从视觉体验升级为"多感官沉浸体验"

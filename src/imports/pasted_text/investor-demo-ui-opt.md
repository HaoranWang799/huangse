2. 通过 AI 剧本 / 场景 / 音频增强使用体验
3. 承接模板商城、会员订阅、裂变和社区预览
4. 让投资人一眼看懂：这是“硬件联动 + AI生成 + 商业化 + 内容生态”的产品

## 三、重点页面与具体优化要求

### 1）飞机杯交互页面（最核心页面，必须重点优化）
目标：
让用户和投资人一眼看懂“内容节奏”和“飞机杯硬件动作”是联动的，而不是普通遥控器页面。

请在这个页面里加强以下设计：
- 进度条不是普通进度条，而是有高低起伏、强弱变化的动态进度条
- 进度条上有明显的节奏节点、高潮节点、强刺激节点
- 用户可以选择频率、强度、节奏，并且看起来像真的在控制硬件
- 需要明显体现“和硬件交互的感觉”
- 页面中心需要一个强视觉主体，可以是设备抽象动效、波形、运动节奏图、上下冲程感、实时反馈动画
- 显示：
  - 设备连接状态
  - 电量
  - AI 模式 / 手动模式
  - 当前阶段
  - 倒计时
  - 强度值
  - 频率值
- 风格不要做成普通健身 App，要做成“成人向智能硬件控制中枢”

关键词：
硬件联动、沉浸感、强反馈、强节奏感、飞机杯控制台

---

### 2）AI 剧本交互引导页面（第二核心页面）
目标：
让用户感觉这不是死模板，而是“可以输入自己的想法，AI 在生成”。

请重点优化：
- 不要让用户完全手动输入 prompt
- 在输入区增加关键词提示词 / 标签 chips / 快捷组合
- 提示词示例：
  - 角色类型
  - 场景类型
  - 强度风格
  - 时长
  - 声线
- 可以做成：
  - 点击标签自动填充
  - 多选组合后生成完整 prompt
  - 快速推荐模板
- 页面上要让人感受到“AI基础生成感”
- 结果页或生成中状态要有：
  - 生成中 loading
  - “AI 正在根据你的偏好生成”
  - 生成完成后的内容卡片

关键词：
Prompt 引导、低门槛、AI生成感、不是手动写作文

---

### 3）商业化信息前置（必须前置，不要藏太深）
目标：
投资人和用户在 Demo 里马上看到“怎么赚钱”。

请在剧本输入页 / 结果页 / 会员入口中体现：
- 本月剩余免费额度
- 已使用次数
- 付费额度
- 付费方式入口
- 明确的会员升级入口
- 普通用户只能体验短时长内容（例如 5-10 分钟）
- 付费用户可以体验更长时长内容（例如 30 分钟）
- 可以出现：
  - 免费版 / Plus / Premium 对比
  - “升级后解锁更长剧本 / 更强模式 / 更高质量内容”
- 商业化露出要自然，但一定要前置，不要只放在深层页面

关键词：
订阅、额度、升级、付费时长差异、商业闭环

---

### 4）商城分类与筛选重构
目标：
让商城不再是散乱内容堆叠，而是结构清晰、适合消费的模板商城。

请优化：
- 顶部 tab：推荐 / 热门 / 最新 / 会员专区
- 筛选维度：
  - 角色
  - 场景
  - 强度
  - 时长
- 每个模板卡片要包含：
  - 封面
  - 标题
  - 标签
  - 热度或下载量
  - 是否会员专属
  - 试看 / 预览入口
- 页面结构更像高端内容消费平台，而不是普通列表页

关键词：
商城重构、分类清晰、筛选明确、可消费、可转化
